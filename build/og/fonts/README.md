# OG Image Fonts

## Font: Mona Sans (Static TTF)

**Files:**

- `Mona-Sans-Regular.ttf` — weight 400, used for body text in OG images
- `Mona-Sans-Bold.ttf` — weight 700, used for headings in OG images

**Source:** GitHub's Mona Sans v2.0.27 static TTFs  
Downloaded from: https://github.com/github/mona-sans/releases/download/v2.0.27/mona-sans-static-v2.0.27.zip

**License:** SIL Open Font License 1.1 (OFL-1.1)  
See: https://github.com/github/mona-sans/blob/main/LICENSE

**Usage in satori (Task 2):** pass `family: "Mona Sans"` in the fonts array.
